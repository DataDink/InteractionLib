Attributes
action
 
data-action
The request uri
method
 
data-method
The method to be used: get, post, put, delete, patch, etc
target
 
data-target
The element that will receive response events
Events
ajax-before
Fires before an ajax request is sent and allows the request to be cancelled and resent
ajax-success
Fires after an ajax request has completed successfully
ajax-failure
Fires after an ajax request has completed unsuccessfully
ajax-after
Fires after and ajax request has completed
Description

Responds to submit or ajax-submit events to trigger an ajax form submission.

This behavior can be added to a regular form or any other element containing named inputs. The inputs will be serialized and sent with the specified method to the specified action/uri. The serialized input data will be sent as a part of the request uri or in the request body depending on which method verb is used.

The result of the request will be sent along with the ajax-success, ajax-failure, and ajax-after events which can be picked up by an ajax-frame or other behavior listening to the specified target element(s).

Example
HTML
<form data-behavior="ajax-form"
         action="http://my-service"
         method="get"
         target="#frame">

         <input type="search" name="query" />
         <button type="submit">Send</button>
</form>

<div  data-behavior="ajax-frame"
         id="frame">
         Enter your search to see results...
</div>
