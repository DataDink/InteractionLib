ajax-failure

The ajax-failure event is fired after an ajax request has completed with a failed status code.

event.detail

The event object passed to listeners of this custom event will have a detail field on it with the following data

form
The source element associated with the ajax request
action
The ajax request URI
query
The serialized data sent with the ajax request
method
The verb used by the ajax request
status
The numeric response code of the ajax response
type
A string value specifying the type of the response data
response
The response content
