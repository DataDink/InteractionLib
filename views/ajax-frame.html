Attributes
data-ajax-listen
A Context Selector pointing to the element(s) to listen for ajax-success events on
data-ajax-target
A Context Selector pointing to the element(s) that ajax content will be sent to
data-ajax-mode
The method used for placing content
Modes
FILL
Ajax content will replace all existing content in the target element(s)
APPEND
Ajax content will be inserted after existing content in the target element(s)
PREPEND
Ajax content will be inserted before existing content in the target element(s)
Description

Responds to ajax-success events to trigger an ajax form submission.

This behavior will typically be set on a div or other block element where ajaxed content is expected to be inserted in the document. Any other part of the document responsible for making ajax request (e.g. ajax-form) can send an ajax-success event to the ajax-frame element to have the HTML response content displayed.

Example
HTML
<form data-behavior="ajax-form"
         action="http://my-service"
         method="get"
         target="#frame">

         <input type="search" name="query" />
         <button type="submit">Send</button>
</form>

<div  data-behavior="ajax-frame"
         id="frame">
         Enter your search to see results...
</div>
