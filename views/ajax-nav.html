Attributes
href
 
data-href
The uri that will be loaded
target
 
data-nav-target
A Context Selector refering to the element(s) that will receive the ajax-success event
data-nav-event
Specifies the event that will trigger the navigation
Events
ajax-success
If the content is successfully loaded this event will be fired on the specified target element(s)
Description

Responds to click or other specified event to trigger an ajax load of content specified by an href or data-href attribute. The loaded content will trigger an ajax-success event that can be directed to an ajax-frame using a target or data-nav-target attribute.

HTML
<nav>
   <a data-behavior="ajax-nav" target="#view" href="views/my-view.html" />
   <a data-behavior="ajax-nav" target="#view" href="views/my-other-view.html" />
</nav>

<div id="view" data-behavior="ajax-frame"></div>

You can also add the ajax-nav behavior to an element that contains multiple href/data-href elements.

HTML
<nav data-behavior="ajax-nav" data-nav-target="#view">
   <a href="views/my-view.html" />
   <a href="views/my-other-view.html" />
</nav>

<div id="view" data-behavior="ajax-frame"></div>
