Context Selectors

Context Selectors add to regular query selectors the ability to select parents, children, and siblings relative to the element queried from.

Example
HTML
<section>
   <article>
      <div data-target="parent>>section"></div>
   </article>
</section>

In this example the query selector "section" is prepended with "parent>>". This will select the first parent of the div that matches "section".

Contexts
parent
Selects the first matching parent element
parents
Selects all matching parent elements
child
Selects the first matching descendant element
children
Selects all matching descendant elements
prev
Selects the first previous matching sibling element
prevAll
Selects all previous matching sibling elements
next
Selects the first following matching sibling element
nextAll
Selects all following matching sibling elements
Extension

You can also invoke a Context Selector directly off of any DOM element in your code using the "contextSelector" extension.

Example
javascript
var outerDiv = document.querySelector('div.inner').contextSelector('parent>>div.outer');
