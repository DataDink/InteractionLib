ajax-before

The ajax-before event is fired prior to sending an ajax request.

event.detail

The event object passed to listeners of this custom event will have a detail field on it with the following data

form
The source element associated with the ajax request
action
The ajax request URI
query
The serialized data sent with the ajax request
method
The verb used by the ajax request
cancel
Set this to false to cancel the request
resubmit
A function that will send/resend the configured ajax request
